## Main-File Analysis Method

Processing steps (per frame):
1. Read frame from main cine.
2. Local background estimate with box blur (`kernel=91`).
3. Enhance dark spray by `enhanced = local_bg - frame`.
4. Otsu threshold on ROI; build binary spray mask.
5. Remove tiny connected components (`min area = 9 px`).
6. Compute Minkowski metrics:
   - Area (`area_px`)
   - Perimeter (`perimeter_px`)
   - Euler characteristic (`euler_chi`)
7. Normalize by ROI for densities (`area_fraction`, `perimeter_density`, `euler_density`).

Run settings:
- Frame size: `2048 x 800`
- Source FPS: `7600`
- Frames analyzed: `3801` (stride 2 from 7601 source frames)
- Time window analyzed: `0.0 to 1.0 s`
